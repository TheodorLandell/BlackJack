# Blackjack-app

Webbaserad blackjack med två lägen: slumpkort och kameraläge med YOLOv8.

## Starta projektet

**Backend**
```bash
cd backend
pip install -r requirements.txt
python app.py
```

**Frontend**

Öppna `frontend/index.html` i webbläsaren.

## Dokumentation

Se mappen `docs/` för promptlogg och övrig dokumentation.
